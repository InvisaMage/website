InvisaMage

Hello, I am InvisaMage.
I am 16 year old human male who likes web technologies and assorted games.
I got the name, InvisaMage, when I was creating a name for my RuneScape character.
I created this website for entertainment and to organize my projects in one place.
My Minecraft skin, as shown above, is armour called "Rune Guthix" from RuneScape.



Creation

This website was hand coded using Notepad++.
This website was tested on Mozilla Firefox Developer Edition.

Git Repository

This website is also backed up on BitBucket and is currently private.
If I get enough requests to make it public, I will most likely do so.
I use SourceTree as my Git GUI.



Information

The font used is called "Ubuntu-R" which is default on the OS called Ubuntu.
This website was tested on Mozilla Firefox and may not work correctly on other browsers.
This website has a few "Easter Eggs" embedded into it. Have fun finding them.
The Clock at the top of every page is in 24 hours.
The Counter on the Homepage displays how many times you've visited the homepage.
