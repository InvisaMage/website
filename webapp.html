iOS Web Application

This page explains how to Add this website to your Home Screen on iOS.
Don't worry, the steps are listed below and there are screenshots.
This project is to help make iOS the user experience on my website more enjoyable.

Instructions

On Safari, go to http://www.invisamage.cf and tap the icon with the arrow pointing away from the box.



Next, tap "Add to Home Screen".



Finally, tap "Add".

Congratulations, you can now use this website as a web application.
You can move it around as you would any other application.
Please note that most things are broken.
The links open in a separate window in Safari.
