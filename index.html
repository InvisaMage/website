Recent News

April 30, 2016

Improved page loading times.
The Terminal is no longer loaded on page load.
Use Shift + T to load the assets and ` to initiate the Terminal.

Added the "load" command to the Terminal.
Allows you to load assets whenever you'd like.

Added the "media" command to the Terminal.
Allows you to control media playback on a page (mostly the Homepage).

Added the "modal" command to the Terminal.
Allows you to dispaly any modal on any page.

The snowstorm and water Easter eggs have been changed as well.
You must now load their assets before you can enable them.
Improved dropdown menus on smaller displays.
