2018


October 12, 2018

Content and Date Jump menus are now fixed when the page is scrolled far enough.
Scrolling to anchors should no longer clip content.
Removed achievement from items in the achievement local storage database (as it was redundant).
This change will cause issues with achievement progress thus resetting progress is recommended.
The legal banner is now an alert.

Minor improvements to the Terminal.
Added bg command to fix background color of the Terminal not being applied.
The close command now allows the animation to be played.


September 14, 2018

Content and Date Jump menu font size has been reduced.
Renamed Purple Rain Achievement to Vulcanic Ash (achievement method is still the same).
Fixed Toggle Terminal setting disable logic.
Fixed Terminal background color bug.

Updated Bootstrap to version 4.1.3.


July 13, 2018

Alerts are now shown on every page.
Events and the Information banners now use PNotify.
Some notifications are now clickable rather than using buttons for actions.
Renamed some files just for the heck of it.
Changed Terminal load key combo to Alt + `.
This was done to prevent typing issues as Shift and Space are used together frequently.

Updated jQuery Terminal to version 1.18.0.


May 25, 2018

PNotify is now being used for alerts/notifications.
This comes with better mobile support and smoother animations compared to bootstrapGrowl.
The alert command on the Terminal now interfaces with this.

The "Cookie" banner has been replaced with the "Legal" banner.
This is now the only place to accept the Terms & Conditions and Privacy Policy.

Updated Bootstrap to version 4.1.1.
Removed the "Silent Mode" from Settings.
The footer has been tweaked.


April 28, 2018

Search on mobile now uses a modal for better usability.
Added new commands and major command tweaks to the Terminal.
The requests command returns unique and total requests to the site.
The libraries command returns the current version of JavaScript libraries used.
The media command will now only function if you're on the homepage and audio is present.
The modal command now prevents the user from opening multiple modals and closing hidden ones.
The anchor command has new syntax and the ability to list all the IDs on the page.
The snowstorm command has more help information.

Updated localForage to version 1.7.1.
Updated Keypress to version 2.1.5.
Updated Konami-JS to version 1.6.0.

Minor tweaks.

Updated Bootstrap to version 4.1.0.
